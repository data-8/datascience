datascience.tables.Table.rows¶

property Table.rows¶

Return a view of all rows.

Returns:

list-like Rows object that contains tuple-like Row objects

Example:

>>> t = Table().with_columns({
...     'letter': ['a', 'b', 'c', 'z'],
...     'count':  [  9,   3,   3,   1],
...     'points': [  1,   2,   2,  10],
... })
>>> t.rows
Rows(letter | count | points
a      | 9     | 1
b      | 3     | 2
c      | 3     | 2
z      | 1     | 10)
