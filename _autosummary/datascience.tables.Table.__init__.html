datascience.tables.Table.__init__¶

Table.__init__(columns=None, labels=None, formatter=<datascience.formats.Formatter object>)[source]¶

Create a table from a list of column values or dictionary of sequences.

>>> letters = ['a', 'b', 'c', 'z']
>>> counts = [9, 3, 3, 1]
>>> points = [1, 2, 2, 10]
>>> t = Table([letters, counts, points], ['letter', 'count', 'points'])
>>> print(t)
letter | count | points
a      | 9     | 1
b      | 3     | 2
c      | 3     | 2
z      | 1     | 10

For other ways to initialize a table, see Table.from_rows(), Table.from_records(), and Table.read_table().

Kwargs:
columns (None, list, or dict): If None, an empty table is

created.

If a list, each element in columns is another list containing the values for a column in the order the columns appear in labels.

If a dict, each key is a label of a column; each values is the column’s values as a list.

labels (list): A list of column labels. Must be specified if
columns is a list, and must be None if columns is a dict.
formatter (Formatter): An instance of Formatter that
formats the columns’ values.
Returns:
A new instance of Table.
Raises:
AssertionError:
  • labels is specified but columns is not.
  • columns is a dict but labels are specified.
  • columns is a list but labels are not specified.
  • The length of labels and the length of columns are unequal.
