datascience.tables.Table.first¶

Table.first(label)[source]¶

Return the zeroth item in a column.

Args:

label (str) – value of column label

Returns:

zeroth item of column

Example:

>>> t = Table().with_columns({
...     'letter': ['a', 'b', 'c', 'z'],
...     'count':  [  9,   3,   3,   1],
...     'points': [  1,   2,   2,  10],
... })
>>> t.first('letter')
'a'
